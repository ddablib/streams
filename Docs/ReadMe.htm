Description

This class library provides some classes that can be used to extend the functionality of Delphi's TStream classes.

Also included in the download are demo programs and a user guide.

» Contents

Compatibility

The classes depend on method overloading and are therefore only compatible with Delphi 4 and later.

» Contents

Demo Projects

Two demo projects are provided:

  • IStreamWrap demonstrates how to use the classes from the PJIStreams unit that implement IStream interfaces for wrapped streams.
  • StreamWrap program demonstrates how to derive a class from the TPJSteamWrapper class. The demo implements a class that can read and write strings from and to a stream.

» Contents

Installation

Simply extract the files from the zip file and store in the required location on your hard disk. Once compiled, if the .dcu files are placed in a folder on your Delphi search path then the .pas files are not required - simply use the units as required.

The demo files are stored in the Demos sub-folder - open the .bpg file in Delphi to compile and examine the demos.
